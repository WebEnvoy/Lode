# Lode

Lode 是 WebEnvoy 的站点知识、站点能力、任务封装与模板资产库。

它为 WebEnvoy Core 提供可复用的网站能力资产；WebEnvoy Core 解释这些资产，并通过 Harbor 获取执行身份与浏览器 Runtime 来完成真实网站读写任务。

## 仓库角色

Lode 负责沉淀和维护：

- 站点知识；
- 站点能力；
- 原子动作；
- 任务封装；
- 官方模板；
- 能力测试样例；
- 能力版本与失效标记。

# Lode 定位

Lode 是 WebEnvoy 的能力资产库。

它不是普通代码包仓库，而是 WebEnvoy 用来沉淀、演化和分发可复用网站能力的地方。

## 承载内容

Lode 承载：

- 站点知识：页面结构、入口路径、接口形状、字段结构、登录态判断、风控状态识别等；
- 站点能力：搜索、读取详情、读取评论、检查账号状态、上传媒体、发布内容、修改内容、删除内容等；
- 原子动作：打开页面、等待状态、点击、填写、上传、提取字段、调用页面内请求、验证结果等；
- 任务封装：将多个站点能力组合成可复用流程；
- 模板：面向常见业务场景的官方任务模板和示例配置。

## 与其他仓库的关系

- WebEnvoy Core 消费 Lode 中的能力定义；
- Harbor 提供执行身份和浏览器 Runtime；
- Lode 不应依赖 Harbor 的内部实现；
- 用户私有任务和私有参数默认不进入公共 Lode。
